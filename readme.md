# Recipes Website

A small static recipe website built as part of The Odin Project fundamentals curriculum.

## Live Website

View the site here: [sergi-fernandez.github.io/odin-recipes](https://sergi-fernandez.github.io/odin-recipes/)

## Purpose

This repository is part of my technical-fluency practice as a Product Manager. The goal was to build confidence with the structure of simple static websites: HTML documents, internal links, image assets, basic styling, and page-to-page navigation.

## Technologies

- HTML
- CSS

## What This Project Demonstrates

- Basic semantic page structure
- Navigation between a homepage and individual recipe pages
- Image handling for static web pages
- External source links for recipe references
- Simple layout, spacing, and visual styling decisions

## Notes

This is a learning project, not a product case study. I keep it public because it shows the foundation of my coding practice. My main PM-facing work is better represented by my portfolio and product case-study materials.
